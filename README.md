# eightqueens

## <span style="color:orange">**Instructions**</span>


Write a program that prints the solutions to the [eight queens puzzle](https://en.wikipedia.org/wiki/Eight_queens_puzzle).

Recursivity must be used to solve this problem.

It should print something like this :

```console
15863724
16837425
17468253
.
.
.
```

Each solution will be on a single line.
The index of the placement of a queen starts at 1.
It reads from left to right and each digit is the position for each column.
The solutions will be printed in ascending order.

You can use one of this coding languages to solve the problem

### <span style="color:yellow">**Python**</span>

### <span style="color:yellow">**Js**</span>
